# Как работать с кодом

## Проверка стиля кода

В репозитории настроена автоматическая проверка кода с помощью `flake8`.

### Как это работает

- При каждом **push** или **pull request** в ветку `main` запускается проверка.
- Если всё хорошо — рядом с коммитом будет зелёная галочка ✅.
- Если есть ошибки — красная ❌.

### Что делать, если проверка упала

1. Перейди на вкладку **Actions** в репозитории GitHub.
2. Нажми на запуск с красным крестиком.
3. Найди секцию **Run flake8** — там будет написано, что не так.

Чаще всего ошибки связаны с:
- Лишними пробелами
- Слишком длинными строками
- Неиспользуемыми импортами

### Как проверить код локально (по желанию)

Если хочешь проверить код до пуша:

```bash
# Установить flake8
pip install flake8
```

# Проверить код
```bash
flake8 .
```
